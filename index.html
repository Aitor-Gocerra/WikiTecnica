Programación

Conceptos Clave

La programación es el proceso de diseñar, codificar, depurar y mantener el código fuente de programas de computadora. Implica una serie de conceptos fundamentales que son la base para construir cualquier aplicación. [cite: 6]

Tipos de Datos

Los tipos de datos son la clasificación de los datos que se pueden utilizar en un programa, lo que afecta cómo se almacenan y manipulan. [cite: 6]

  • Enteros (Integer): Números sin decimales, como 5, -100.
  • Flotantes (Float/Double): Números con decimales, como 3.14, -0.5.
  • Cadenas de Texto (String): Secuencias de caracteres, como "Hola mundo", "Programación".
  • Booleanos (Boolean): Valores de verdad, true o false.

Estructuras de Control

Las estructuras de control determinan el flujo de ejecución de un programa. [cite: 6]

  • Condicionales (if, else if, else): Permiten ejecutar bloques de código basados en una condición.
  • Bucles (for, while, do-while): Permiten repetir bloques de código varias veces.

Funciones

Las funciones son bloques de código reutilizables que realizan una tarea específica. Ayudan a organizar el código y a evitar la duplicación. [cite: 6]

Sintaxis Básica (en pseudocódigo o JavaScript)

Aquí se muestran ejemplos de la sintaxis básica para los conceptos clave. [cite: 7]

Condicionales (Pseudocódigo)


SI (condicion ES VERDADERO) ENTONCES
    // hacer algo
SINO SI (otra_condicion ES VERDADERO) ENTONCES
    // hacer otra cosa
SINO
    // hacer algo diferente
FIN_SI

Bucle For (JavaScript)


for (let i = 0; i < 5; i++) {
    console.log("Iteración número: " + i);
}

Función (JavaScript)


function sumar(a, b) {
    return a + b;
}
let resultado = sumar(5, 3); // resultado será 8
console.log(resultado);

Ejemplos Simples de Código Comentado


// Este es un ejemplo de una función simple en JavaScript [cite: 7]
function saludarUsuario(nombre) {
    // La función toma un argumento 'nombre'
    console.log("¡Hola, " + nombre + "!"); // Imprime un saludo en la consola
}

// Llamada a la función
saludarUsuario("Ana"); // Esto imprimirá "¡Hola, Ana!" en la consola

// Ejemplo de un bucle 'while' para contar hasta 3

    console.log("Contador: " + contador);

}

Conceptos de Redes, Software y Hardware

Redes

Una red informática es un conjunto de equipos conectados por medio de cables, señales, ondas o cualquier otro método de transporte de datos, que comparten información, recursos y servicios. [cite: 8]

  • IP (Protocolo de Internet): Identificador único para cada dispositivo en una red.
  • HTTP/HTTPS: Protocolos para la comunicación web.
  • Routers y Switches: Dispositivos que dirigen el tráfico en una red.

Software

El software es la parte lógica de un sistema informático, que incluye el conjunto de programas, instrucciones y reglas informáticas. [cite: 8]

  • Sistemas Operativos: Como Windows, macOS, Linux.
  • Aplicaciones: Programas diseñados para tareas específicas (navegadores web, procesadores de texto).
  • Lenguajes de Programación: Herramientas para escribir software (Python, Java, C++).

Hardware

El hardware se refiere a los componentes físicos y tangibles de un sistema informático. [cite: 8]

  • CPU (Unidad Central de Procesamiento): El "cerebro" del ordenador.
  • RAM (Memoria de Acceso Aleatorio): Almacenamiento temporal para datos y programas en ejecución.
  • Dispositivos de Almacenamiento: Discos duros (HDD), unidades de estado sólido (SSD).
  • Periféricos: Teclado, ratón, monitor, impresora.

Bases de Datos

Las bases de datos son sistemas organizados para almacenar, gestionar y recuperar grandes cantidades de información de manera eficiente. [cite: 9, 10]

Tipos de Bases de Datos

Existen diversos modelos y tipos de bases de datos, cada uno con sus propias características y aplicaciones. [cite: 9]

  • Bases de Datos Relacionales (SQL): Organizaciones de datos en tablas con filas y columnas, y relaciones entre ellas. Ejemplos: MySQL, PostgreSQL, SQL Server. [cite: 9]
  • Bases de Datos NoSQL: Ofrecen flexibilidad y escalabilidad, sin el esquema fijo de las relacionales. Ejemplos: MongoDB (documentos), Cassandra (columnas anchas), Redis (clave-valor). [cite: 9]

Ejemplos de Sentencias SQL Básicas

SQL (Structured Query Language) es el lenguaje estándar para interactuar con bases de datos relacionales. [cite: 9]

SELECT

Se utiliza para recuperar datos de una o más tablas. [cite: 9]


-- Selecciona todas las columnas de la tabla 'usuarios'
SELECT * FROM usuarios; [cite: 9]

-- Selecciona solo el nombre y el correo electrónico de la tabla 'clientes'
SELECT nombre, email FROM clientes WHERE edad > 30;

INSERT

Se utiliza para añadir nuevas filas de datos a una tabla. [cite: 9]


-- Inserta un nuevo producto con nombre y precio
INSERT INTO productos (nombre, precio) VALUES ('Teclado', 25.99); [cite: 9]

-- Inserta un nuevo usuario con todos los campos
INSERT INTO usuarios (id, nombre, apellido, email) VALUES (1, 'Juan', 'Perez', 'juan.perez@example.com');

UPDATE

Se utiliza para modificar datos existentes en una tabla.


-- Actualiza el precio de un producto específico
UPDATE productos SET precio = 29.99 WHERE nombre = 'Teclado';

DELETE

Se utiliza para eliminar filas de datos de una tabla.


-- Elimina un usuario con un ID específico
DELETE FROM usuarios WHERE id = 5;

Conceptos Clave

  • Clave Primaria (Primary Key): Es una columna o conjunto de columnas en una tabla que identifica de forma única cada fila en esa tabla. No puede contener valores nulos y sus valores deben ser únicos. [cite: 10]
  • Clave Foránea (Foreign Key): Es una columna o conjunto de columnas en una tabla que se refiere a la clave primaria de otra tabla. Establece una relación entre dos tablas. [cite: 10]
  • Relaciones: Conexiones lógicas entre tablas, generalmente establecidas mediante claves primarias y foráneas, que definen cómo los datos de una tabla se relacionan con los datos de otra. [cite: 10]
    • Uno a Uno (1:1): Una fila en la tabla A se relaciona con una sola fila en la tabla B.
    • Uno a Muchos (1:N): Una fila en la tabla A se relaciona con muchas filas en la tabla B.
    • Muchos a Muchos (M:N): Muchas filas en la tabla A se relacionan con muchas filas en la tabla B (requiere una tabla intermedia).
  • Esquema: La estructura o el diseño de una base de datos, incluyendo las tablas, columnas, tipos de datos, relaciones y restricciones.

Lenguajes de Marcas

Los lenguajes de marcas son sistemas para anotar documentos de una manera que sea sintácticamente distinguible del texto. Son utilizados para definir la estructura y presentación de la información. [cite: 3]

Componentes HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura del contenido web. [cite: 3, 10]

  • Elementos: Son la base de HTML, formados por una etiqueta de apertura, contenido y una etiqueta de cierre (ej. <p>Párrafo</p>).
  • Etiquetas (Tags): Palabras clave que definen cómo un navegador web debe formatear y mostrar el contenido. Ejemplos: <html>, <head>, <body>, <h1>, <p>, <a>, <div>.
  • Atributos: Proporcionan información adicional sobre los elementos HTML y se colocan dentro de la etiqueta de apertura (ej. <a href="pagina.html">).
  • Estructura Semántica: HTML5 introdujo etiquetas semánticas para dar significado a la estructura del contenido, como <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>. [cite: 3]

Ejemplo de Estructura HTML Básica


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Página</title>
</head>
<body>
    <header>
        <h1>Título de la Página</h1>
    </header>
    <main>
        <section>
            <p>Contenido principal aquí.</p>
        </section>
    </main>
    <footer>
        <p>&copy; 2025</p>
    </footer>
</body>
</html>

CSS (Cascading Style Sheets)

CSS es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento escrito en HTML. Permite controlar el diseño, los colores, las fuentes y otros aspectos visuales. [cite: 4, 10]

  • Selectores: Apuntan a los elementos HTML a los que se les aplicarán estilos (ej. body, h1, .clase, #id).
  • Propiedades y Valores: Las propiedades definen qué aspecto del elemento se va a estilizar (ej. color, font-size) y los valores son la configuración para esa propiedad (ej. blue, 16px).
  • Modelos de Caja (Box Model): Define cómo los elementos HTML son tratados como cajas con propiedades de margen, borde, relleno y contenido.
  • Diseño Responsivo: Uso de CSS para que el diseño de la web se adapte a diferentes tamaños de pantalla (mediante media queries).
